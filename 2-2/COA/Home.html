COMPUTER ORGANIZATION AND ARCHITECTURE

Computer organization and architecture focus on the structure, design, and functionality of computer systems. Computer organization deals with how components like the CPU, memory, and input/output devices are organized and interconnected to form a functional computer system. It involves studying topics such as instruction set architecture, data representation, and memory hierarchy. On the other hand, computer architecture is concerned with the design principles and performance characteristics of computer systems. It includes designing instruction sets, optimizing performance, and evaluating trade-offs between different hardware components. Together, computer organization and architecture provide the foundation for understanding how computers work at both the hardware and software levels, enabling efficient and effective system design and development.

Major Features of Computer Organization and Architecture

  • Instruction Set Architecture (ISA).
  • Memory Hierarchy.
  • Processor Design.
  • Input/Output Systems.
  • Parallelism and Concurrency.
  • System Interconnection.
  • Performance Evaluation.
  • SSecurity and Protection.

COURSE OUTCOMES

  • Understand the basic organization of computers, different instruction formats and addressing modes. [L2]
  • Comprehend the 8086 microprocessors [L2]
  • Interpret the types of memory, CPU-memory connection, and microprogrammed control unit [L2]
  • Compare various CPU- I/O data transfer modes, and asynchronous data transfer techniques [L2]
  • Describe parallel processing, Flynn’s classification and multiprocessor architectures [L2]

SYLLABUS

UNIT-I

Instruction: Instruction Definition, instruction cycle, flow chart for instruction cycle, instruction storage, types of instruction formats (Zero, one, two and three address). Addressing modes: mode field, implied, immediate register, register direct, register indirect, auto increment, decrement, indexed, relative, base address mode, Numerical examples and problems.

UNIT-II

CPU-Organization: 8086 –CPU –Block diagram and pin diagram, minimum and maximum mode, General purpose registers; segment register and generation of 20 bits address, segmentation of main memory, systems bus, Types of flags.

UNIT-III

Memory Hierarchy, Main memory, memory address map, memory connection to CPU; Auxiliary memory, Magnetic disks, Magnetic tapes; cache memory, hit and miss ratio, direct, associative and set associative mapping; Micro-programmed control: control memory, address sequencing.

UNIT-IV

I/O interface: I/O Bus and Interface modules, I/O versus Memory Bus, isolated vs Memory-mapped I/O. Asynchronous data transfer-strobe control, Hand shaking; Modes of Transfer: Example of programmed I/O, interrupt-initiated I/O. Daisy-Chaining priority. DMA: DMA Controller, DMA Transfer, Intel 8089 IOP.

UNIT-V

Pipeline and Vector Processing: Parallel Processing, Pipelining, Arithmetic Pipeline, Instruction Pipeline, RISC Pipeline, Vector Processing, Array Processor. Multi Processors: Characteristics of Multiprocessor; Interconnection structures: Time Shared common bus, multiport memory, crossbar switch, multi-stage switching network; Introduction to Flynn’s classification: SISD, SIMD, MISD, MIMD (Introduction).
